⬇ Download extension (ZIP) Installation guide

Install in 4 steps

  1. Download the ZIP and unzip it anywhere on your computer.
  2. Open chrome://extensions and enable Developer mode (top right).
  3. Click Load unpacked and select the unzipped extension folder.
  4. Pin Repository Tech Radar to the toolbar and start scanning.

Works on Chrome, Edge, Brave and other Chromium browsers. No account required for free mode.
